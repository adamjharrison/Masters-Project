# Masters-Project
Code used for Masters Project to test the saddle point approximation of the SOC couplings between TBFs in Generalised Ab Initio Multiple Spawning

The SPA1_SOC_model function within the SPA1.f90 file is purely included to show the main code produced for this project, however this will not run by itself as it is integrated into the larger OpenFMS nonadiabatic dynamics codebase. More details on how this function is interfaced within OpenFMS can be seen in the merged commit here: https://github.com/ispg-group/openfms/commit/bd061019bf5d5c313afa324edec250b0ef357f05
